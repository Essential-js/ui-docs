<section>
	<h3>Setup</h3>

    <p>To start using the toast component you simply have to import the `Toasts` component and define the absolute position where you want the toast list to be displayed.</p>
    <p>Also, to fire toast you only have to use the toast object and using the type that you want to fire execute it, each type of toast is a method that receives as first parameter the message that the toast must have and as second parameter the duration of this one in screen in milliseconds.</p>

</section>

<CopyableCode>
{`
import React from 'react';
import {Toasts, toast} from '@essential-js/ui/toast';

function App() {

    function displayToast() {
                  /// Messsage     // Duration
        toast.info('Awesome toast!', 2000)
    }

return (

        <Toasts
            position={{
            bottom: '2rem',
            right: '2rem',
            }}
        />
    )

}
`}

</CopyableCode>
